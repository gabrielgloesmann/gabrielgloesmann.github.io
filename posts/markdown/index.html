Introduction to Markdown
November 8, 2021 · Gabriel Glösmann

Markdown is a markup language with which it is possible to create formatted text using a text editor. The use of Markdown ranges from blogs, web pages to documents. With the help of pandoc, markdown pages can be converted into LateX and Word documents.

Writing#

Headers:

# H1
# H2
## H3
## H4
## H5
## H6

Italics:

*asterisks* or _underscores_

Bold:

**asterisks** or __underscores__

Strikethrough:

~~Scratch this~~

Lists#

Ordered:

1. Item
2. Item
⋅⋅1. Sub Item
⋅⋅2. Sub Item

Unordered:

* Item
* Item
⋅⋅* Sub Item
⋅⋅* Sub Item

Links#

Inline:

[I'm an inline-style link](https://www.google.com)

[I'm an inline-style link with title](https://www.google.com "Google's Homepage")

Files:

[I'm a relative reference to a repository file](../blob/master/LICENSE)
