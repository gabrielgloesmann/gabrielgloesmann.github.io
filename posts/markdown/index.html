Introduction to Markdown
July 17, 2023 · Gabriel Glösmann

Markdown is a lightweight markup language used for formatting text. It allows you to add formatting elements like headers, lists, emphasis (bold and italic), links, images, and more to plain text documents. Markdown is commonly used for creating web content, writing documentation, or formatting text in various applications and platforms.

Writing#

Headers:

# H1
# H2
## H3
## H4
## H5
## H6

Italics:

*asterisks* or _underscores_

Bold:

**asterisks** or __underscores__

Strikethrough:

~~Scratch this~~

Lists#

Ordered:

1. Item
2. Item
⋅⋅1. Sub Item
⋅⋅2. Sub Item

Unordered:

* Item
* Item
⋅⋅* Sub Item
⋅⋅* Sub Item

Links#

Inline:

[I'm an inline-style link](https://www.google.com)

[I'm an inline-style link with title](https://www.google.com "Google's Homepage")

Files:

[I'm a relative reference to a repository file](../blob/master/LICENSE)
