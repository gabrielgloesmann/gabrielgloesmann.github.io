Docker

Docker is a virtualization platform at the operating system level. The individual programs run in so called containers. It is used for developing, running and shipping applications. JSON

JSON is a compact data format represented in an easy-to-read text form, which is used for data exchange between applications. The advantage is that it is independent of the programming language. LateX

LateX is a document creation program which, unlike other programs, uses markup tags to specify the formatting of the document. LateX is very common in the academic world because it is very suitable for the representation of mathematical formulas. Markdown

Markdown is a markup language with which it is possible to create formatted text using a text editor. The use of Markdown ranges from blogs, web pages to documents. With the help of pandoc, markdown pages can be converted into LateX and Word documents.
