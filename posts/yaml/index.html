YAML

Introduction to YAML
July 20, 2023 · Gabriel Glösmann

YAML stands for “YAML Ain’t Markup Language.” It is a human-readable data serialization format used for representing structured data in a format that is easy for both humans and machines to understand. YAML is often used for configuration files and data exchange between different programming languages. It uses indentation and simple syntax to define data structures such as lists, dictionaries, and scalars.

Data Types#

- Number
- String
- Boolean
- Array
- Object

Comments#

# This is a comment.
foo: bar # This is an inline comment.

Numeric Types#

foo : 1230.15
bar : 1230150.0
foo: 12345
bar: 0x12d4
plop: 023332

Arrays#

items: [ 1, 2, 3, 4, 5 ]
names: [ "one", "two", "three", "four" ]
