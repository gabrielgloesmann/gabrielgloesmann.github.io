Introduction to Git
November 8, 2021 · Gabriel Glösmann

Git is a software which is used for version management of files. The inventor and founder of Git was Linus Torvalds.

Workflow#

Repository consits of three trees:

  1. Working Directory: holds the actual files
  2. Index: Staging area
  3. Head: points to the last commit

Workflow:

Working Directory -> Add -> Index -> Commit -> Head

Create repository#

Create a git repository:

git init

Version#

git --version

Configuration#

Set username:

git config --global user.name "YOUR_USERNAME"

Check username:

git config --global user.name

Set E-Mail:

git config --global user.email "your_email_address@example.com"

Check E-Mail:

git config --global user.email

Clone repository#

Create folder where to clone the code:

cd Project

Clone the project:

git clone https://git.presslabs.net/user/mysite-dev

Go to the repository directory:

cd mysite-dev

Show repository content:

ls -lh

Status#

git status

Add files to index#

Add one file to Index:

git add file01.txt

Add all files to index:

git add -A

Undo file:

git reset file01.txt

Commit files to head#

Commit files to head (SHA):

git revert 6ffa13ce232966bbff8091a1ca
git push

Pushing the changes#

Pull changes from others:

git pull

Push changes:

git push
