Introduction to Git
July 21, 2023 · Gabriel Glösmann

General#

Git is a version control software.

Workflow#

The repository consists of trees:

  1. Working Directory
  2. Staging Area
  3. Local Repo
  4. Remote Repo

![[Git/git.png]]

Settings#

Version:

git --version

Status:

git status

Set the username:

git config --global user.name "YOUR_USERNAME"

Set the E-Mail:

git config --global user.email "your_email_address@example.com"

Repository#

Create a git repository:

git init

Clone a project:

git clone https://git.github.net/user/mysite-dev

Add one files to the staging area:

git add file01.txt
git add .

Commit files to the local repo:

git commit -m "This is a commit."

Push the changes to the remote repo:

git push

Pull changes from the remote repo:

git pull

Create a new branch:

git checkout -b branchname

Push the new branch to the remote repo:

set stream git push --set-upstream origin branchname
