LateX

LaTeX is a document preparation system commonly used for typesetting technical and scientific documents. It provides a set of macros and commands that allow users to create high-quality documents with complex structures, mathematical equations, tables, figures, citations, and references. LaTeX files are typically compiled into PDF documents. It is widely used in academic and scientific circles for writing research papers, theses, and technical reports. Kubernetes

General Kubernetes is an open-source container orchestration platform that automates the deployment, scaling, and management of containerized applications. It provides a framework for managing distributed systems and ensures that containers are deployed and run efficiently across clusters of computers. Kubernetes abstracts the underlying infrastructure and enables efficient scaling, load balancing, and fault tolerance for applications deployed in containers. JSON

JSON (JavaScript Object Notation) is a lightweight data-interchange format commonly used for transmitting data between a server and a client, or between different parts of an application. It is based on a subset of JavaScript programming language syntax and represents data as key-value pairs. JSON is human-readable, easy to parse and generate, and supported across various programming languages.
