General#

Python programming refers to the practice of writing computer programs using the Python programming language. Python is a high-level, versatile, and easy-to-read programming language known for its simplicity and readability. It provides a wide range of libraries and frameworks that make it suitable for various purposes, including web development, scientific computing, data analysis, artificial intelligence, and more. Python code is typically executed using an interpreter, making it an interpreted language. Python’s clean syntax and extensive community support have contributed to its popularity among beginners and experienced developers alike.

Installation#

Install python:

brew install python

Install the pyenv to manage the python versions:

brew install pyenv

Install the python version with pyenv:

pyenv install 3.11

Set the python version:

pyenv global 3.11

Check versions:

pyenv versions

Set the macOS PATH for OhMyZSH:

echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc

Then run this:

echo -e 'if command -v pyenv 1>/dev/null 2>&1; then\n  eval "$(pyenv init --path)"\n  eval "$(pyenv init -)"\nfi' >> ~/.zshrc

Virtual Environment#

Create a virtual environment:

make venv

Start the virtual environment:

. venv/bin/activate
