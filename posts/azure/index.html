Introduction to Azure
July 12, 2023 · Gabriel Glösmann

General#

Azure is a cloud computing platform and service provided by Microsoft. It offers a wide range of cloud-based services such as virtual machines, databases, storage, networking, artificial intelligence, and more. Azure allows individuals and organizations to build, deploy, and manage applications and services on a global network of Microsoft-managed data centers. It supports various programming languages, frameworks, and operating systems, and it is commonly used for developing, testing, and deploying applications, as well as for data storage, backup and recovery, and analysis.

Services#

IaaS#

IaaS in Azure refers to Infrastructure as a Service. It is a cloud computing model where Azure provides virtualized computing resources over the internet. With Azure IaaS, you can create and manage virtual machines, storage, networks, and other infrastructure components without the need to purchase and maintain physical hardware. It gives you flexibility and scalability while allowing you to focus on deploying and managing your applications.

PaaS#

In Azure, PaaS stands for “Platform as a Service.” It is a cloud computing model that provides a platform for developers to build, deploy, and manage applications without the complexity of infrastructure management. Azure PaaS offerings include services such as Azure App Service, Azure Functions, Azure SQL Database, and Azure Logic Apps. These services provide pre-configured computing environments and services, allowing developers to focus on coding and application development rather than managing the underlying infrastructure.

SaaS#

In Azure, SaaS stands for “Software as a Service.” It refers to a cloud computing model where software applications are provided over the Internet as a service. With Azure SaaS, users can access and use software applications hosted and managed by a third-party provider, such as Microsoft, without the need for installation or maintenance on their own computers. Examples of Azure SaaS offerings include Microsoft Office 365, Dynamics 365, and Azure DevOps.

ARM#

Azure Resource Manager (ARM) is the deployment and management service for resources in Microsoft Azure. It provides a consistent management layer that allows you to deploy, manage, and organize Azure resources, such as virtual machines, storage accounts, and databases, as a group or individually.
