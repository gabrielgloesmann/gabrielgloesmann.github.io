JSON

Introduction to JSON
November 8, 2021 · Gabriel Glösmann

JSON is a compact data format represented in an easy-to-read text form, which is used for data exchange between applications. The advantage is that it is independent of the programming language.

Data Types#

- Number
- String
- Boolean
- Array
- Object

Filename Extensions#

.json

Examples#

{
  "firstName": "John",
  "lastName": "Smith",
  "isAlive": true,
  "age": 27,
  "address": {
    "streetAddress": "21 2nd Street",
    "city": "New York",
    "state": "NY",
    "postalCode": "10021-3100"
  },
  "phoneNumbers": [
    {
      "type": "home",
      "number": "212 555-1234"
    },
    {
      "type": "office",
      "number": "646 555-4567"
    }
  ],
  "children": [],
  "spouse": null
}
