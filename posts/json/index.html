Introduction to JSON
July 1, 2023 · Gabriel Glösmann

JSON (JavaScript Object Notation) is a lightweight data-interchange format commonly used for transmitting data between a server and a client, or between different parts of an application. It is based on a subset of JavaScript programming language syntax and represents data as key-value pairs. JSON is human-readable, easy to parse and generate, and supported across various programming languages. It is often used in web APIs, configuration files, and as a standard data format for data exchange.

Data Types#

- Number
- String
- Boolean
- Array
- Object

Example#

{
  "firstName": "John",
  "lastName": "Smith",
  "isAlive": true,
  "age": 27,
  "address": {
    "streetAddress": "21 2nd Street",
    "city": "New York",
    "state": "NY",
    "postalCode": "10021-3100"
  },
  "phoneNumbers": [
    {
      "type": "home",
      "number": "212 555-1234"
    },
    {
      "type": "office",
      "number": "646 555-4567"
    }
  ],
  "children": [],
  "spouse": null
}
