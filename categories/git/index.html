Git

General Git is a version control software. Workflow The repository consists of trees: Working Directory Staging Area Local Repo Remote Repo Settings Version: git --version Status: git status Set the username: git config --global user.name "YOUR_USERNAME" Set the E-Mail: git config --global user.email "your_email_address@example.com" Repository Create a git repository: git init Clone a project: git clone https://git.github.net/user/mysite-dev Add one files to the staging area: git add file01.txt git add ....
